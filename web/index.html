ACPUL - Another CPU Language - a{};



About

Удобный и простой язык программирования.
Прост в изучении, минимален и гибок.
Разработан специально для ACPU и разработке на планшетах с ограниченными возможностями. Совместим с Google Glass.
Главная цель получать более качественные кросс-платформенные приложения при уменьшении времени разработки, улучшении архитектуры (сохраняя принцип 80/20) и мобильности в сочетании простоты, доступности и максимальной производительности.

ACPU - Animation CPU

Анимационный процессор, аналог классического хардварного процессора. Аналогично аппаратным портам ввода-вывода, которые обеспечивают связь с различными устройствами, в ACPU существуют виртуальные порты, которые обеспечивают связь с любым объектом в виртуальной среде (touch-display, cloud, media, etc.). Это позволяет связывать разные технологии универсальным методом.

ACPUEM - ACPU Execution Module

Модуль исполнения, Virtual Machine. Полностью виртуализирован и может работать в любой среде: iOS, OSX, Android, Linux, Windows, Adobe Flash, Java, HTML5, Unity3d.
Внутренняя структура настолько универсальна, что может быть реализована даже в аппаратном чипе на основе SoC с кластерной микроархитектурой для достижения максимальной производительности.
